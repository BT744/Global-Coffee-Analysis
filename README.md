# Global Coffee Analysis (Python)

## Overview

This project is to provide a exploratory data analysis on various products offered in the coffee shop across four countries. 

The goal is to explore patterns in spending habits, consumption levels for different products, preferred payment method and if weather will affect the transactions. 

## Skills Demonstrated

Summary of the dataset
Data cleaning by identifying missing values and remove columns and rows with null values
Filter by amount spent
Scatterplot
Bar graph
Box-and-whisker plot
